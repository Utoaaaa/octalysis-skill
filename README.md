# Octalysis Skill

This is the publishable skill package distilled from:

- 《遊戲化實戰全書》 by 周郁凱 Yu-kai Chou
- Three local Octalysis course transcript notes

The result is not a gamification tricks library. It is a general motivation architecture for diagnosing and designing human behavior across habits, learning, work, products, communities, education, health behavior, onboarding, B2B tools, and AI-agent workflows.

## Contents

- `SKILL.md` — root routing skill
- `skills/` — eight executable skills
- `assets/` — reusable motivation-design asset library
- `README.md` and `INDEX.md` — package guide and navigation

## Executable Skills

- `motivation-diagnosis`
- `strategy-dashboard`
- `core-drive-intervention`
- `journey-stage-mapping`
- `feedback-loop-design`
- `mission-narrative-design`
- `case-pattern-transfer`
- `motivation-ethics-review`

## Important Publishing Note

Only this inner `octalysis-skill/` directory is intended to be published. The parent workspace keeps local-only distillation materials such as `source/`, `books/`, `.agents/`, and `distillation-checks/`.

Do not copy original source files into this package. The distilled assets and skills are designed to stand alone without publishing source texts or audit work files.
