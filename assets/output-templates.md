## First-Level Octalysis Scorecard

Use this when a user asks for a broad Octalysis analysis, but keep it qualitative unless reliable data exists.

```text
| Drive | Strength: missing/weak/medium/strong/overused | Evidence | Stage | Intervention implication |
| CD1 | | | | |
| CD2 | | | | |
| CD3 | | | | |
| CD4 | | | | |
| CD5 | | | | |
| CD6 | | | | |
| CD7 | | | | |
| CD8 | | | | |
```

Interpretation rules:

- Do not optimize all eight drives.
- High CD6/CD7/CD8 is not automatically good; check agency and fatigue.
- High CD2/CD4/CD6 may indicate extrinsic dependency.
- Low CD1/CD3/CD5 often explains why behavior feels transactional, controlled, or isolated.
- Scores without evidence are hypotheses, not findings.

## Explicit / Implicit Mode Choice

Use explicit mode when visible game-like language, roles, quests, levels, or collectibles will clarify progress, identity, or participation.

Use implicit mode when the audience needs seriousness, privacy, professional trust, accessibility, or when visible game framing would feel patronizing.

Always state:

- why the chosen mode fits the actor,
- what would go wrong in the opposite mode,
- how the mode affects trust, agency, and long-term motivation.
